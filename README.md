# TSHUKI COLLECTOR
A curated display of collectible anime figures — formerly "The Display Case." This project showcases animated, interactive figure cards with tilt and flip effects.

Features
- Responsive grid gallery
- Keyboard-accessible flip interactions
- Scroll-triggered entrance animations
- Special animated effects for highlighted figures (e.g., Luffy, Itachi)

How to run locally
1. Clone the repo
2. Open index.html in your browser, or serve with a static server:
   - Python: `python -m http.server 8000`
   - Node: `npx http-server .`
